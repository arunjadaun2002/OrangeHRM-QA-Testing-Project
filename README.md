# OrangeHRM-QA-Testing-Project

# OrangeHRM Manual Testing Project

## Project Overview
This repository contains a complete Manual Testing project for the OrangeHRM application focusing on the Login Module.

The project was created to practice real-world QA processes including:
- Test Planning
- Test Scenario Writing
- Test Case Design
- Bug Reporting
- Jira Bug Tracking
- Selenium Automation Testing

---

# Application Tested
OrangeHRM Demo Application

Module Tested:
- Login Module
- Forgot Password Functionality
- Validation Messages
- UI Responsiveness

---

# Testing Types Performed
- Functional Testing
- UI Testing
- Negative Testing
- Validation Testing
- Basic Responsive Testing
